# Privacy Policy for Zynox Discord Bot

**Effective Date:** January 1, 2025

## 1. Introduction

Zynox ("we", "us", or "our") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our Discord bot ("Zynox" or "the Bot").

## 2. Information We Collect

### 2.1 Automatically Collected Information
- **Server Data**: Server ID, channel IDs, role IDs, member counts
- **User Data**: User IDs, usernames, discriminator, join dates
- **Message Data**: Message content (for moderation features), message IDs, timestamps
- **Audit Log Data**: Server events, moderation actions, user actions
- **Voice Data**: Voice channel join/leave events (no audio content)

### 2.2 Stored Data
We store the following data in our database:
- Server configurations and settings
- User warning counts and moderation history
- Whitelist and blacklist entries
- Logging preferences and channel settings
- Custom messages and automation rules

### 2.3 Data Retention
- Data is retained as long as the bot is in your server
- Data is deleted when the bot leaves your server
- Moderation logs may be retained for audit purposes

## 3. How We Use Your Information

### 3.1 Bot Functionality
- Provide moderation and protection features
- Process commands and automation rules
- Generate logs for server administration
- Maintain whitelists and permission systems

### 3.2 Service Improvement
- Analyze usage patterns to improve features
- Debug and fix technical issues
- Monitor bot performance and stability

## 4. Data Sharing and Disclosure

### 4.1 Third-Party Services
- **Discord API**: We interact with Discord's API to function
- **Logging Services**: Webhook-based logging to your designated channels
- **No External Sharing**: We do not sell, trade, or share your data with third parties

### 4.2 Legal Requirements
We may disclose information if required by law or to protect rights and safety.

## 5. Data Security

### 5.1 Security Measures
- Encrypted database storage
- Secure API communications
- Access controls and authentication
- Regular security audits

### 5.2 Data Breaches
In case of a data breach, we will notify affected users promptly.

## 6. User Rights

### 6.1 Data Access
Server administrators can request data deletion or export.

### 6.2 Data Deletion
- Remove the bot from your server to delete associated data
- Contact us for manual data removal requests

### 6.3 Opt-out
- Disable specific features to limit data collection
- Use ignore commands to exclude channels/users

## 7. Children's Privacy

The Bot is not intended for children under 13. We do not knowingly collect personal information from children under 13.

## 8. International Data Transfers

Your data may be processed on servers located outside your country. We ensure appropriate safeguards are in place.
